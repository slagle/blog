This widget displays badge with a link to a LinkedIn profile page.

== Description ==
This plugin adds simple widget that will display a badge with a description and a link to a LinkedIn public profile page. Plugin fully supports new Widget model introduced in WordPress 2.8 and allows multiple instances.

Includes several translations: English, Serbian, Lithuanian, French, Spanish, Russian, Portuguese, Danish, German, Dutch, Norwegian.
More details about translations in the languages.txt file.

== Installation ==
* Upload `gd-linkedin-badge` folder to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Add widget to sidebar and set it to your liking

== Configuration ==
Here is the list of all configuration options:
* Title:
  The title of the widget.
* Text:
  Used for ALT for a LinkedIn image.
* Description:
  Text above the LinkedIn badge.
* LinkedIn URL:
  Url to a LinkedIn public profile page
* Badge Style:
  Badge to display.
